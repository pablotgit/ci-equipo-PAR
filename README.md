# ci-equipo-PAR
##Repositorio para tarea colaborativa "Automatización de CI con GitHub Actions en un Proyecto Multilenguaje"

## Integrantes
- Rodrigo Emmanuel Torres
- Andres Fernandez
- Pablo Andres Tonelli

## Descripción
Aplicación simple en Node.js que realiza operaciones matemáticas básicas 
(suma, resta, etc.) con pruebas automatizadas.

## Lenguaje
Node.js

## Workflow
Se ejecuta en cada push y pull request.
Instala dependencias con npm y ejecuta tests con Jest.

## Cómo ejecutar

npm install
node src/index.js

## Cómo correr tests

npm test
